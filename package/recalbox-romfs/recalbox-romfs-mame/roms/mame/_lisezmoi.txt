## RECALBOX - SYSTEM MAME ##

Placez ici vos roms mame.

Les roms doivent avoir l'extension ".zip"

Par défaut, recalbox utilise le core libretro mame2003. Les roms doivent donc provenir du romset mame 0.78

Sont également présents dans Recalbox:
- core libretro imame4all: romset mame 0.37b5
- core libretro mame2010 (non disponible pour rpi0/1): romset mame 0.139
- émulateur standalone advancemame: romset mame 0.106

Si vous souhaitez utiliser un core/émulateur différent de celui par défaut, vous devrez modifier votre fichier recalbox.conf selon les informations présentes dans cette page :
https://github.com/recalbox/recalbox-os/wiki/recalbox.conf-%28FR%29

Vous pouvez utiliser clrmamepro disponible sur http://mamedev.emulab.it/clrmamepro/
et utiliser le fichier .dat dans le répertoire clrmamepro pour vérifier la version de vos roms.

Fichiers spéciaux pour le core mame2003 :
- Ajoutez vos fichiers samples dans le répertoire /recalbox/share/bios/mame2003/samples/
- Mettez à jour le fichier hiscore.dat dans /recalbox/share/bios/mame2003/ si vous souhaitez avoir les derniers highscores (http://highscore.mameworld.info/download.htm)
- Téléchargez cheat.dat dans /recalbox/share/bios/mame2003/ pour activer les codes de triche (http://cheat.retrogames.com/download/cheat081.zip)
- Téléchargez history.dat dans /recalbox/share/bios/mame2003/ pour activer en jeu le menu history (http://www.arcade-history.com/index.php?page=download)
