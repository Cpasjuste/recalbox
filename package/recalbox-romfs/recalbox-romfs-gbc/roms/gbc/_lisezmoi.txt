## RECALBOX - SYSTEM GBC ##

Placez ici vos roms game boy et game boy color.

Les roms doivent avoir l'extension ".gb/.gbc/.zip"

Vous pouvez télécharger les roms game boy à l'adresse :
http://www.planetemu.net/roms/nintendo-game-boy

Vous pouvez télécharger les roms game boy color à l'adresse :
http://www.planetemu.net/roms/nintendo-game-boy-color
