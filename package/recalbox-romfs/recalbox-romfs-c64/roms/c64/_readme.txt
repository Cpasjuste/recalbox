## RECALBOX - SYSTEM COMMODORE 64 ##

Put your Commodore 64 roms in this directory.

Rom files must have a ".d64/.d71/.d80/.d81/.d82/.g41/.g64/.t64/.x64/.tap/.prg/.p00/.crt/.zip" extension.

This system allows to use compressed roms on .zip.
But, it is only an archive. Files inside the .zip must match with extensions mentioned above.
Each .zip file must contain only one compressed rom.


Mapping gamepad :

Start = gui menu
Select = switch mouse/joystick
Y = virtual Keyboard
L2 = switch Joy1/Joy2
