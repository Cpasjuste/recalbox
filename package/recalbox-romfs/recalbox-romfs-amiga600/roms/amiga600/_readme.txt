## RECALBOX - SYSTEM AMIGA 600 ##

Put your amiga 600 roms in this directory.

Rom files must have a ".adf/.uae" extension.
.uae is used only for whdload functionnality

This system does not support compressed roms in .zip.
This systems requires bios to work.

To enter the GUI, press Start
To exit the emulator press Hotkey

Documentation can be found here : 
https://github.com/recalbox/recalbox-os/wiki/Amiga-emulation-on-Recalbox-(EN)
